# Skill — Context Discovery

Objective:
Reduce initial ambiguity through minimal, high-value questions.

---

# RULES

* Maximum 3 questions per iteration
* Prioritize structural questions
* Avoid vague questions
* Do not start implementation without sufficient context

---

# PRIORITY QUESTIONS

1. What problem do you want to solve?
2. What input does the system receive?
3. What output should it produce?

Optional:

* constraints
* success criteria
* reference systems

---

# OUTPUT

Generate:

`.ai/templates/problem-definition.md`

or an equivalent structured markdown file.

---

# FINAL OBJECTIVE

Transform:

* vague ideas
* conversations
* ambiguous prompts

into:

* structured context
* reusable definitions
