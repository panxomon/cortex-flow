# docs/analysis

Reportes de evaluación derivados de la ejecución de flows.

Vive aquí:
- análisis de drift entre intención y realidad
- diffs (e.g. `oas-diff.md`)
- evaluaciones de deuda técnica
- reportes de cobertura, dependencias obsoletas, vulnerabilidades

No vive aquí: contratos fuente de verdad (van en `definitions/contracts/`).
